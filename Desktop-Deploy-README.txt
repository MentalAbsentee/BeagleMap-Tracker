BeagleMap - Deploy Folder on Desktop
=====================================

This folder contains the cloud-ready deployment files for BeagleMap.

CONTENTS:
---------
- index.html     : Main application
- app.js        : Core functionality
- vercel.json    : Vercel deployment config
- netlify.toml   : Netlify deployment config
- _headers       : HTTP headers for deployment

QUICK START:
-----------
1. Open this folder
2. Double-click index.html
3. Or run a local server:
   python -m http.server 8000
   Then visit: http://localhost:8000

CLOUD DEPLOYMENT:
----------------
Upload this entire folder to:

VERCEL (Recommended):
1. Go to vercel.com
2. Import project
3. Upload this folder
4. Deploy instantly

NETLIFY:
1. Go to netlify.com
2. Drag & drop this folder
3. Deploy automatically

GITHUB PAGES:
1. Upload to GitHub repository
2. Enable Pages in settings
3. Access via github.io URL
